# 💪 건강관리 프로그램
**사용자에게 적절한 운동이나 음식을 추천해주고, 체중, 권장섭취량, 칼로리 등을 계산해주는 건강관리 프로그램**

* 2019년 1학기 데이터베이스(2019.03 ~ 2019.06)
* 2인 1조 프로젝트
* PHP, JavaScript, MySQL, Apache
* **주요 기능**: 회원가입, 게시판 CRUD, 사용자 정보를 기반으로 한 운동/음식 추천, 체중/권장섭취량/칼로리 계산
